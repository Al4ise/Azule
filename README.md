# Azule

## Installation
```zsh
echo "export AZULE=~/azule" >> ~/.zprofile
git clone --recursive https://github.com/Al4ise/Azule $AZULE
chmod +x $AZULE/install
$AZULE/install
```

## Usage
```
azule
```

## Updating
```
update-azule
```
